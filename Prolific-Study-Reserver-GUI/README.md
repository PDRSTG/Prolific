# Prolific Study Reserver GUI

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)]()
[![Tampermonkey](https://img.shields.io/badge/Userscript-Tampermonkey-brightgreen.svg)]()
[![Status](https://img.shields.io/badge/Status-Active-blue.svg)]()

A browser userscript for automatically attempting to reserve Prolific studies using randomized timing, GUI controls, and reactive state awareness.

## Features
- Randomized auto-clicking of the **Start Study** button.
- GUI interval configuration (F9).
- Start/Stop hotkeys (F7/F8).
- Smart detection of Full, Paused, Returned, and High Demand states.
- Success detection when the button changes to “I'm finished”.

## Installation
1. Install Tampermonkey.
2. Create a new script.
3. Paste in the contents of `Prolific Study Reserver GUI.js`.
4. Save.

## Usage
- Press **F9** to set intervals.
- Press **F7** to start.
- Press **F8** to stop.
